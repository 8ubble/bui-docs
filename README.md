bui-docs
===

BUI 控件库的文档
- [控件库首页](http://www.builive.com/)
- [应用代码](https://github.com/dxq613/bui-default)

## 文件结构

- tools : 文件打包，以及生成文件的工具
- api : api文档

## 重新生成文档库
- 将[bui](https://github.com/dxq613/bui)放置到跟bui-docs在同一个目录下
- 执行 tools/jsduck/run.bat
